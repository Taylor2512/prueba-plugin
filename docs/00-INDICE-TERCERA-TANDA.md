# Tercera tanda de documentación técnica

Esta tercera tanda profundiza a nivel de **archivo, contrato, flujo y decisiones de implementación** del proyecto `sisad-pdfme`, con foco en los puntos que más valor aportan para mantenimiento, venta, escalabilidad y migración futura hacia una plataforma PDF reutilizable.

1. `01-DESIGNER-ENGINE-Y-BUILDER-PROFUNDO.md`
2. `02-CANVAS-OVERLAYS-Y-CICLO-DE-INTERACCION.md`
3. `03-RIGHTSIDEBAR-LISTVIEW-Y-DETAILVIEW-PROFUNDO.md`
4. `04-SCHEMACONNECTIONS-Y-CONTRATOS-DE-DATOS.md`
5. `05-COLABORACION-ARQUITECTURA-OPERATIVA.md`
6. `06-SCHEMAS-Y-REGISTRY-GUIA-DE-EXTENSION.md`
7. `07-GENERATOR-CONVERTER-Y-RUNTIME-DE-VISUALIZACION.md`
8. `08-PLAN-DE-MIGRACION-A-PLATFORM-PDF.md`

## Objetivo de esta tanda

A diferencia de la primera y segunda entregas, esta tanda:

- baja a nivel de archivo y módulo concreto;
- describe contratos internos y dependencias cruzadas;
- propone estrategias de endurecimiento, extensibilidad y branding;
- sirve como base para auditoría, refactor guiado y empaquetado comercial.
